{# ── Rules ─────────────────────────────────────── #}
Import Rules
  • Rows with a duplicate AFSN are skipped by default. Tick Update existing records to overwrite them instead.
  • Rows with a duplicate telephone number belonging to a different person are always skipped.
  • Rows with missing required fields or invalid values are also skipped.
  • Personnel QR codes are generated automatically after each row is saved.
  • The first row is always treated as the header row (Excel). Google Sheets uses the first row as the header automatically.
  • Blank rows are ignored.
  • The optional photo column accepts a Google Drive share link. The file must be shared publicly (Anyone with the link → Viewer).
