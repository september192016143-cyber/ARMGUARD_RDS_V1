Install SSL Certificate

Installing this certificate removes the "Not Secure" warning and shows a padlock in your browser. Do this once on each device.

  1. 1 Click Download Certificate below — the file armguard-selfsigned.crt will save to your Downloads folder.
  2. 2 Double-click the downloaded .crt file. A Windows certificate dialog will open.
  3. 3 Click Install Certificate.
  4. 4 Select Local Machine → click Next.
    Windows may ask for administrator permission — click Yes.
  5. 5 Choose Place all certificates in the following store → click Browse → select Trusted Root Certification Authorities → click OK.
  6. 6 Click Next → Finish. You will see a security warning — click Yes to confirm.
  7. 7 Close all browser windows completely, then reopen and go to https://192.168.0.11. The padlock will now be green with no warning.
Only needed once per device. If the certificate is renewed in the future, you will be notified here.
