Place Arial Nova .ttf font files here for server deployment.
The update-server.sh script will install them to /usr/share/fonts/truetype/arial-nova/.

Required files (copy from C:\Windows\Fonts\ on a Windows machine with Arial Nova):
  ArialNova.ttf
  ArialNovaBd.ttf    (Bold)
  ArialNovaI.ttf     (Italic)
  ArialNovaBdI.ttf   (Bold Italic)
  ArialNovaLight.ttf
  ArialNovaLightI.ttf

These files are NOT committed to the repository (commercial font).
